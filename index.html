React app

Github link

In my free time I have been learning the frontend framework 'React'. By combining my interest in health and some different React functions I created a full stack web app that takes your intake of calories and the amount you excerized at any given time and stores them in a database. The stored data is then graphed based on your net calories per day. I chose to use the NoSQL database MongoDB and Express for my API development. This is still a work in progress.
